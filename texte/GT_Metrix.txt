GT Metrix - Stichpunkte
    - Automatisiertes Tool zum testen der Ladegeschwindigkeiten von Webseiten
    - es ist eine gratis und eine bezahlte version verfügbar
    - features der gratis version 
        - core web vitals -> sind ein wichtiger bestandteil des google suchalgorithmus 
        - GT Metrix gibt eine Report mit folgenden Inhalten
            - Performance -> wie gut performed die website aus der Sicht es Nutzers
            - Structure Score -> zeigt wie gut die Webseite für optimale Performance gebaut ist
        - Speed Visualisation -> eine intervallweise Erfassung Ihres Seitenaufrufs mit Fähnchen, die Ihre verschiedenen Leistungskennzahlen während des Seitenaufrufs markieren
        - Top Issues -> eine List mit Strukturproblemen, die die größten Leistungseinbuße haben
        - Page Details -> Aufteilung welche Requests getätigt wurden - inklusive wie viele und die größe der Requests
        - GT Metrix Sektion -> Tips zur Optimierung der Website basierend auf der Analyse
        - Einzelne Tabs für Performance, Sturktur usw. -> für eine detailiertere Übersicht der Analyse
        - mit 7 Locations weltweit kann man die Performace der Website in verschiedenen Standorten testen
        - automatische tägliche, wöchentliche oder monatliche tests machen um eine trendanalyse zu erhalten
            - sehen wie die Entwicklung der ... ist
                - web vitals 
                - GTmetrix Grade, Performance / Structure Scores
                - HTML / Total Page Size, Number of Requests
        - Wasserfall Diagramm
            - identifiziere lange oder defekte Requests
            - nachvollziehen des Ladeprozesses
        - weitere Features
            - Tests mit und ohne Adblocker
            - Verbinungsdrosselung -> wie ist die Analyse bei schlechterer Internetverbindung
            - Benachrichtigung wenn Performace unter ein gegebenes Level fällt
            - Video Playback -> was sieht der User beim Laden der Webseite

    - features der bezahlten version
        - alle Features die die gratis version bietet +
        - Mobile Testing -> teste deine Webseite auf mehr als 40 Mobilgeräten -> iPhones, Galaxies, Pixels und mehr
        - 15 weitere globale Standorte zum testen der Webseite -> zusätzlich zu den 7 der gratis version
        - stündliche Überwachung der Performance -> erlaubt eine detailierte Einsicht zu welcher Tageszeit Performance Einbuße zu verzeichnen sind
        - unbegrenzte Anzahl an Tags -> Sortiere und Kategorisiere Fehlerstellen und Reporte
        - mehr Analyseoptionen und Features 
            - Bildschirmauflösung
            - individuelle Internetgeschwindigkeiten
            - Reporte werden länger aufbewahrt -> bessere Einblicke in die vergangene Leistung
            - eigener DNS -> Ändern Sie die Hostnamen/IP-Adressen für Testentwicklungs-/Staging-Umgebungen
        - Höchste Priorität der Nutzeranfragen Warteschlange
        - Weltweite Überwachung
        - API-Funktionalität
        - Vollständiger PDF-Report zum Herunterladen
